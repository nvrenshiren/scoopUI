## 1. 目的

为用户提供一个图形界面,集中读取并即时编辑 Scoop 自身(`scoop config`)支持的全部配置项,免去记忆命令行键名与手改 `config.json`。配置按类别分区呈现,每项改动即时生效。本页面覆盖的是 Scoop 自身配置体系,与产品自有的语言/主题/安装配置(P09)相互独立。

## 2. 功能清单

- 功能 1:F20 Scoop 配置可视化编辑 —— 读取 Scoop 全部配置项当前值,按类别分区展示;用户修改某项后即时写入 Scoop(`scoop config`),或将某项恢复为默认(`scoop config rm`)。
- 功能 2:F19 错误处理与友好提示 —— 配置读取失败、写入失败时以业务语言提示;写入失败时界面回滚到写入前的值。

## 3. 页面流转

### 3.1 入口

从 P03 主界面壳侧边栏"Scoop 配置"导航项进入(`view: "config"`)。进入前随 Boot Sequence 进入主界面后的初始加载读取一次当前配置。

### 3.2 内部流转

读取并展示全部配置项当前值(未设置项以其默认值作占位提示)→ 用户修改某项 → 即时写入 Scoop → 写入成功该项保留新值、失败回滚并提示 → 用户可对已设置项点"恢复默认"清除该项。目录类危险项在写入/恢复前二次确认。

### 3.3 出口

用户在侧边栏切换到其它视图即离开本页;配置改动已即时落盘,无需保存动作。读取失败时页面呈现错误提示,可在环境恢复后重新进入。

## 4. 交互说明

- 场景 1:查看全部配置
  - 触发:用户从 P03 进入本页。
  - 呈现:配置项按类别分区(下载器 / 网络与代理 / 更新策略 / 安装与工具 / 目录 / 令牌与密钥),每项展示名称、说明、原始键名与当前值控件;已显式设置过的项额外提供"恢复默认"。
  - 结果:用户一览 Scoop 当前配置全貌。
- 场景 2:开关型配置即时切换
  - 触发:用户拨动某开关型配置(如"启用 aria2 下载")。
  - 呈现:开关立即反映新状态,该项即时写入 Scoop。
  - 结果:配置立即生效;该项此后显示"恢复默认"入口。
- 场景 3:枚举型配置切换
  - 触发:用户在枚举型配置(如"Scoop 分支""默认架构""Shim 实现")的候选值间选择。
  - 呈现:选中值高亮,即时写入。
  - 结果:配置立即生效。
- 场景 4:文本/数值型配置编辑
  - 触发:用户在文本或数值输入框修改值并失焦或回车。
  - 呈现:值即时写入;清空输入等同于恢复默认。
  - 结果:配置立即生效。
- 场景 5:敏感令牌编辑
  - 触发:用户编辑令牌类配置(GitHub 令牌 / VirusTotal 密钥)。
  - 呈现:输入内容默认遮罩,可切换显示;值仅在本机读写。
  - 结果:令牌写入本机 Scoop 配置,不在界面明文长期暴露、不外传。
- 场景 6:目录类危险项修改
  - 触发:用户修改目录类配置(根目录 / 全局目录 / 缓存目录)。
  - 呈现:写入前弹出确认,说明"仅修改配置、不搬移已安装数据,改错可能导致 Scoop 找不到已装应用"。
  - 结果:用户确认后才写入;取消则保持原值。
- 场景 7:恢复默认
  - 触发:用户对某已设置项点"恢复默认"。
  - 呈现:该项从 Scoop 配置中移除,回到默认态(危险项同样先确认)。
  - 结果:该项恢复 Scoop 默认行为。
- 场景 8:读取/写入失败
  - 触发:配置读取失败,或某项写入失败。
  - 呈现:读取失败时页面给出错误提示;写入失败时该项回滚到写入前的值并提示。
  - 结果:用户获知失败,且界面状态与 Scoop 实际保持一致。

## 5. 验收要点

- ✅ 用户应能从 P03 侧边栏进入 P11。
- ✅ 页面应覆盖 Scoop `scoop config` 支持的全部配置项,按类别分区。
- ✅ 每项应展示名称、说明与当前值;控件形态应与配置类型匹配(开关/枚举/文本/数值/令牌/只读)。
- ✅ 修改任一项应即时写入 Scoop,无需额外"保存"动作。
- ✅ 写入失败时该项应回滚到写入前的值并给出提示。
- ✅ 已显式设置过的项应提供"恢复默认",未设置项不提供。
- ✅ 目录类配置项修改或恢复默认前应二次确认,并说明不搬移数据的风险。
- ✅ 令牌类配置项应默认遮罩且可切换显示,不外传。
- ✅ 复杂数组/对象类项(如私有主机)可只读展示并提示走命令行编辑,不强行提供图形编辑。
- ✅ 本页只覆盖 Scoop 自身配置,不改变产品自有的语言/主题/安装配置(P09),也不改变软件包、桶或长时任务状态。
- ✅ 全部文案应中英双语齐备,配置项名称/说明随界面语言切换。
