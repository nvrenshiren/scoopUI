# P05 · 浏览与搜索软件包页

## 1. 目的

帮助用户浏览已添加桶范围内的可装软件包,并通过关键字快速定位目标软件包。用户可从本页查看详情并对单个可装软件包发起安装。

## 2. 功能清单

- 功能 1:F01 软件包浏览 —— 展示已添加桶中的可装软件包范围。
- 功能 2:F02 软件包搜索 —— 在已添加桶范围内按关键字定位软件包。
- 功能 3:F04 软件包详情查看 —— 用户选择单个软件包后进入 P07 查看详情。
- 功能 4:F05 软件包安装(单包) —— 用户可对一个可装未装的软件包发起安装。
- 功能 5:F17 安装任务进度反馈 —— 安装过程进入 P08 展示进度与结果。
- 功能 6:F19 错误处理与友好提示 —— 搜索无结果、列表为空、安装失败时给出友好提示。

## 3. 页面流转

### 3.1 入口

本页面从 P03 主界面壳进入,属于 `flows/scoop-gui.md` §1.2 日常主界面流程中的可装软件包浏览与搜索区段。用户从安装流程返回时,引用 `flows/scoop-gui.md` §1.3 的业务结果刷新本页。

### 3.2 内部流转

可装列表展示 → 用户输入或清空查询 → 展示全部可装软件包或匹配结果 → 选中条目 → 进入 P07 查看详情或发起安装 → 安装进入 P08 → 完成后刷新本页。应用包状态口径只引用 `flows/scoop-gui.md` §2.1,长时任务口径只引用 §2.4。

### 3.3 出口

- 选中软件包查看详情时进入 P07,对应 `flows/scoop-gui.md` §1.2 中 C04。
- 对可装软件包发起安装时进入 P08,对应 `flows/scoop-gui.md` §1.3 单包安装流程。
- 搜索无结果、可装范围为空或读取失败时在本页内呈现 P10 子状态,引用 `flows/scoop-gui.md` §1.9。

## 4. 交互说明

- 场景 1:浏览全部可装软件包
  - 触发:用户进入本页且未输入查询内容。
  - 呈现:界面展示已添加桶范围内的全部可装软件包。
  - 结果:用户可滚动浏览、选择详情或安装。
- 场景 2:搜索软件包
  - 触发:用户输入关键字。
  - 呈现:界面展示与关键字匹配的软件包结果。
  - 结果:用户可以从结果中选择目标软件包。
- 场景 3:清空搜索
  - 触发:用户清空查询内容。
  - 呈现:界面恢复展示已添加桶范围内的全部可装软件包。
  - 结果:用户回到浏览状态。
- 场景 4:查看软件包详情
  - 触发:用户选中某个可装软件包。
  - 呈现:打开 P07,展示该软件包详情与安装入口。
  - 结果:用户可关闭返回本页或继续安装。
- 场景 5:安装软件包
  - 触发:用户对某个可装软件包选择安装并确认。
  - 呈现:进入 P08 展示安装过程与结果。
  - 结果:成功后该软件包不再作为可装未装项展示,并应能在已装列表中体现。
- 场景 6:无结果或读取失败
  - 触发:搜索无匹配结果,或可装范围无法读取。
  - 呈现:界面展示 P10 空状态或错误状态。
  - 结果:用户获得清空查询、重试或前往桶管理的下一步动作。

## 5. 验收要点

- ✅ 用户进入本页且查询为空时,应看到已添加桶范围内的全部可装软件包。
- ✅ 用户输入查询后,应只看到与查询匹配的软件包结果。
- ✅ 用户清空查询后,列表应恢复为已添加桶范围内的全部可装软件包。
- ✅ 搜索范围应限于已添加桶,不应展示未添加桶中的软件包。
- ✅ 用户选中软件包后,应能进入 P07 查看详情。
- ✅ 可装未装的软件包应提供安装入口;已装软件包不应在本页被当作可装未装项重复安装。
- ✅ 用户确认安装后,应进入 P08 并看到安装进度与最终结果。
- ✅ 安装成功后返回本页时,该软件包的状态应发生变化,不应继续作为可装未装项展示。
- ✅ 安装成功后,用户应能在 P04 已装软件包列表中看到该软件包。
- ✅ 安装失败时,用户应看到失败原因与重试入口。
- ✅ 搜索无结果时,界面应说明没有匹配项,并提供清空查询或调整查询的动作。
- ✅ 已添加桶范围内没有可装软件包时,本页应显示 P10 空状态而不是空白列表。
- ✅ 读取失败时,本页应显示 P10 错误状态并允许重试。
