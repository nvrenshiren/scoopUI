P10 · 空状态与错误页P10 · Empty & Error States

P10 不是独立顶级页面,而是 P03 / P04 / P05 / P06 的子状态层。当数据为空、读取失败、Scoop 整体不可用、长时任务失败时,接管当前页面内容区。

P10 is not a top-level page. It overlays P03 / P04 / P05 / P06 when data is empty, reads fail, Scoop is unavailable, or long-running jobs fail.
