安装配置

以下配置项与 Scoop 官方安装脚本 install.ps1 一一对应;留空即使用官方默认值。确认后开始安装,安装过程中不可修改;本次配置将保存在本机,重装时可复用。

Scoop 本体与所有软件包的安装位置,默认 %USERPROFILE%\scoop
使用 --global 安装的软件包存放位置,默认 C:\ProgramData\scoop。修改此项需要管理员权限。
安装包下载缓存位置,默认位于安装根目录内的 cache 子目录
通过指定代理下载,例如 http://127.0.0.1:7890;留空则不使用代理
不走代理的主机列表,逗号分隔
代理需要身份验证时填写;留空则不使用凭据
使用 Windows 默认凭据访问代理 (ProxyUseDefaultCredentials)
以管理员权限运行安装 (RunAsAdmin) 勾选后将弹出 UAC 提权窗口执行安装;需要设置全局软件包目录等场景时使用。不勾选则以当前用户身份安装(推荐)。
