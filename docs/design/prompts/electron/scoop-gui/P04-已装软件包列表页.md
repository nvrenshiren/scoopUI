## 1. 页面定位

P04 是 **P03 主界面壳 内的子页面**(从侧边栏"已装"进入,见 PRD §3.1 / 设计系统 §10.1)。**渲染在 P03 骨架内**:

- Sidebar 240px(同 P03,顶部 1px 渐变线 + 选中态 2px primary 微光)
- PageHeader `h-14` + Breadcrumb + 右侧 Refresh icon 按钮
- 主区承载 SubToolbar(主 CTA) + FilterTabs + Table + Footer(分页预留)

原型须先画出 P03 侧边栏 + 顶栏外壳,再在 Content 区画出 P04 主体;**不要重复做 P03 骨架的设计决策**(沿用已审批契约)。

### 3.6 Table(设计系统 §8.6 · 强制 h-10)

**行高强制 40px**,**仅水平分隔线**(无垂直线),sticky thead + body `overflow-auto`。

| 列 | 宽度 | 内容 | 字体 |
|---|---|---|---|
| Name | 32% min-220 | Package 图标(过期时换 AlertTriangle)+ 包名(`font-heading font-medium`)+ 选中态附加 `badge-primary "已选中"` | DM Sans |
| Version | 11% | 版本号 `font-mono text-xs text-muted`(选中行升级 `text-fg`) | **JetBrains Mono** |
| Source | 10% | 桶名 `badge-secondary`(bg-bg-overlay + border-border) | DM Sans |
| Updated | 12% | 日期 `font-mono text-xs text-muted` | JetBrains Mono |
| 状态 | 13% | 过期行:`badge-warning`(bg-warning/15 + text-warning + border-warning/30 + 内嵌 AlertTriangle);正常行:`text-xs text-muted "已装/Installed"` | DM Sans |
| 操作 | 22% min-220 | 三按钮组(过期行)/ 两按钮组(正常行),右对齐 | DM Sans |

**过期行视觉**:`.tr-row.outdated { background-color: var(--warning-soft) }`,行首 14px `AlertTriangle` 图标 + 状态列 Badge。
**选中行视觉**:`.tr-row.selected { background-color: var(--primary-glow-soft) }`,`::before` 左侧 2px primary + `box-shadow: 0 0 10px primary-glow`。**优先级**:选中行覆盖过期行底色(用户态 > 状态态)。
**行内按钮**:

| 过期行 | 正常行 |
|---|---|
| `btn-ghost-c` 详情 + `btn-outline-c` 更新(带 RefreshCw 图标) + `btn-text-destructive-c` 卸载 | `btn-ghost-c` 详情 + 84px 间隔占位(防列宽抖动) + `btn-text-destructive-c` 卸载 |

## 8. 不变量(交给下游 developer 与 QA)

1. 行内 "更新" 按钮仅当 `OutdatedApp.isOutdated === true` 时显示;不存在"未过期但显示可更新"的视觉态。
2. "更新所有过期" 按钮在过期项数为 0 时 `disabled`(`opacity-50 cursor-not-allowed`,设计系统 §8.4),文案保持不变,**不切换为 "暂无过期项"** 文案(避免误导)。
3. P08 在场景 4/5/6 期间,**P04 表格本身不禁用**(设计系统 §8.4 仅约束"按钮 disabled",页面表格可继续滚动);用户在 P04 内可继续查看其他行(操作按钮仍可见,P08 已遮罩底层)。
4. 取消长时操作后,过期行继续显示过期状态(取消 ≠ 失败,见 PRD §5 验收 12)。
5. 选中行视觉(2px primary 微光 + bg-primary/10)必须严格强于过期行视觉(bg-warning/5 + AlertTriangle);当一行同时为过期+选中时,选中态覆盖过期态底色,但保留过期 Badge 与图标。
6. 全部 UI 文案通过 t() 调用,**禁止硬编码中英文案**(设计系统 §8.6)。
7. P08 子对话框为非 modal,`backdrop-filter: blur(2px)` + `background: rgba(0,0,0,0.3)`,允许用户继续浏览主页(与 v1 全 modal 不同)。
