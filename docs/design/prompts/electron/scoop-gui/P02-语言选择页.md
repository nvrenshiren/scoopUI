| 项 | 值 |

## 2. 设计语言定位

- **气质**:首次启动引导,简洁、克制、中性。不堆装饰、不放 hero 图、不用插画。
- **视觉锚点**:大字号主标题(`text-4xl` `font-bold`)+ 中部两张语言卡片 + 底部单一主操作。
- **主色使用**:`--primary` 仅用于:① 选中卡片的边框 + 角标对勾;② 主操作"继续"按钮。其它地方全部走 `--foreground` / `--muted-foreground` / `--border`。
- **状态语义**:选中 = 边框 `--primary` + 背景 `--accent` + 右上角对勾图标;未选中 = 边框 `--border` + 背景 `--card`。
- **不引入**:第三方语言入口、自动翻译入口、语言包导入入口、emoji、彩色插画、Sidebar。

## 4. 区段规范

### 4.1 顶部 Stepper(可选极简)

- 文字形式:`第 1 步 / 共 2 步 · 选择语言` / `Step 1 of 2 · Choose language`
- 字号 `text-xs`,颜色 `--muted-foreground`,**不使用进度条 / 圆点**(避免视觉噪音)
- 距主标题 `mb-2`(8px)

### 4.2 主标题区

- 主标:`text-4xl`(36px)`font-bold`,`text-foreground`
  - 中:`选择界面语言`
  - 英:`Choose your display language`
- 副标(英文用户友好):`text-base`,`text-muted-foreground`,`mt-2`
  - 中:`本设置将决定所有界面文案的显示语言`
  - 英:`This setting determines the language used throughout the interface`
- 距卡片组 `mt-8`(32px)

### 4.3 语言卡片组(核心交互)

- 容器:`flex gap-6`(24px),水平并排,响应式:窗口缩放可保持双列
- 单卡:`w-72`(288px)`h-56`(224px)`p-6`,圆角 `rounded-lg`,背景 `--card`,边框 1px `--border`,阴影 `shadow-sm`
- 卡片内容(自上而下):
  1. **标题**:卡片顶部 `text-xl font-semibold`
     - 中文卡:`中文`;英文卡:`English`
  2. **预览区**:左侧小图标(可选 `Languages` lucide 图标,16px,`--muted-foreground`)+ 4 行示例文案,字号 `text-sm`,行高 `leading-relaxed`,颜色 `--foreground`,各行 8px 行间距
     - 中文预览(必含 4 个 UI 词,展示中文渲染):
       ```
       · 文件
       · 编辑
       · 设置
       · 帮助
       ```
     - 英文预览(必含 4 个 UI 词,展示英文渲染):
       ```
       · File
       · Edit
       · Settings
       · Help
       ```
  3. **底部语言代码**:`text-xs`,`text-muted-foreground`,`mt-auto`(顶到底部)
     - 中文卡:`zh-CN`;英文卡:`en-US`
- **选中态**(场景 2/3):
  - 边框:由 `--border` 切到 `--primary`(2px),用 `ring-2 ring-primary` 表达,避免破坏卡片 1px 边框的几何
  - 背景:`--card` → `--accent`(轻微,保证文字对比度仍满足 WCAG AA)
  - **右上角对勾徽标**:`absolute top-3 right-3`,`rounded-full bg-primary text-primary-foreground`,内嵌 `Check` 图标 16px
  - **键盘焦点态**(仅 Tab 进入时):`focus-visible:ring-2 focus-visible:ring-ring focus-visible:ring-offset-2`,不与选中态冲突(选中态边框 = 2px `--primary`,焦点环 = 2px `--ring`,视觉可区分)
- **悬停态**(未选中时):`hover:border-foreground/40 hover:shadow-md`(微提阴影,提示可点)
- **点击交互**:
  - 整卡可点击(角色:`button`,`role="radio"` 用于单选语义,父容器 `role="radiogroup"`)
  - 点击瞬间触发视觉切换(选中态即时生效,**不需等"继续"按钮**)
  - 已有选中时点击另一卡,旧卡取消选中,新卡选中

### 4.4 主操作按钮

- 位置:卡片组下方,水平居中,`mt-8`(32px)
- 组件:Button `variant="default" size="lg"` → 高度 48px(`h-12`)
- 文案:
  - 中:`继续`;英:`Continue`
- **未选择时**:`disabled` 属性 + `opacity-50 cursor-not-allowed`(由设计系统 §8.4 强制)
- **未选择时 hover 提示**:原生 `cursor-not-allowed`,不弹 Tooltip(避免视觉噪音);`title` 属性可设 `"请先选择语言"` / `"Please select a language"`
- **可选择时**:正常 `hover:bg-primary/90` 提亮,点击触发保存(见 §6)

### 4.5 底部说明

- `mt-4`(16px),`text-xs`,`text-muted-foreground`,居中
- 中:`此次选择可稍后在"设置"中修改`
- 英:`You can change this later in Settings`

## 5. 场景呈现(对应 PRD §4 全部场景)

### 场景 1:首次无语言偏好(默认态)

- 两张卡片均为**未选中**(边框 `--border`,背景 `--card`,无对勾)
- "继续"按钮 `disabled`
- Stepper 显示 `1/2`

### 场景 2:选择中文

- 触发:用户点击中文卡
- 呈现:
  - 中文卡切到**选中态**(2px `--primary` ring + `--accent` 背景 + 右上角对勾)
  - 英文卡保持未选中
  - "继续"按钮启用(可选中态)
- 用户点"继续":
  - 进入场景 2.1:保存中(场景 2.a)
  - 按钮 `disabled` + 文案切到"保存中…" / `Saving…`
  - 整卡片组保持选中态,视觉锁住防二次点击
- 场景 2.1a:保存成功 → 启动序列推进,本页面销毁
- 场景 2.2a(保存失败):见场景 4

### 场景 3:选择 English

- 同场景 2,但选中的是英文卡
- 文案切到英文
- "继续"按钮启用,点击进入保存流程

### 场景 4:保存失败

- 触发:`scoop:prefs:set` 返回 `{ ok: false, error: { code: 'E_PREFERENCES_WRITE_FAILED', ... } }`
- 呈现(**禁用 inline alert**,遵循设计系统 §8.5,使用 Toast):
  - 右下角 Sonner Toast `variant="warning"`(使用 `--warning`)
  - 持续时间 5 秒(设计系统 §7.10 警告档默认)
  - 文案:
    - 中:`语言已即时生效,但本机保存失败,下次启动可能需要重新选择`
    - 英:`Language applied now, but saving failed. You may need to choose again next time.`
- **同时**:页面**不阻塞**,按钮恢复为"继续"态(可选中且可点击),**不视为致命错误**
- 启动序列继续推进(PRD §4 场景 4 明确"启动流程继续推进,不新增第三种语言状态")
- 持久化失败后,主进程应记录结构化日志(开发者关注,UI 不暴露堆栈)

## 6. 关键交互细节(契约级)

| 交互 | 行为 | 备注 |
| --- | --- | --- |
| Tab 键进入页面 | 首个可聚焦元素 = 中文卡(`autoFocus` 或自然 Tab 序) | 遵循设计系统 §8.8 |
| 卡片间 Tab | 中文卡 ↔ 英文卡 ↔ "继续"按钮 | 单选语义,Tab 序列三个可聚焦项 |
| 卡片 Enter / Space | 等价点击,切换选中 | `role="radio"` 键盘约定 |
| Enter 在按钮上 | 等价点击"继续" | 仅当启用时;禁用时浏览器原生拦截 |
| Esc | **不关闭页面**(首次启动无后台页可回,PRD 也不允许回退) | 全局快捷键本页面无响应 |
| 点击"继续"后 | 按钮 disabled + 文案 → "保存中…" / `Saving…` | 防止重复提交,设计系统 §8.4 |
| 选中态切换 | 即时(无 300ms 延迟);仅视觉层 | 后端写入异步进行 |

## 8. 不做什么(契约级排除)

- ❌ 不展示第三方语言(仅 zh-CN / en-US)
- ❌ 不提供自动翻译入口
- ❌ 不提供语言包导入 / 自定义语言
- ❌ 不展示"上次选择"或"默认"项(PRD §3.3 强调仅首次出现)
- ❌ 不使用 inline alert(全部走 Toast / Dialog)
- ❌ 不使用 emoji / 彩色插画
- ❌ 不在选择前展示已选语言(场景 1 双卡皆未选)
- ❌ 不提供"返回"或"上一步"按钮(启动序列单向推进)
- ❌ 不阻塞主线程,保存失败不弹阻塞 Dialog
- ❌ 不暴露错误堆栈
